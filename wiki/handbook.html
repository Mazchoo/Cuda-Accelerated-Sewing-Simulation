Project Documentation

This documentation provides an overview of the modules and their functions available in this project.

Shapers

Shapers are responsible for transforming geometric objects.

Translate

translate(Tuple[float, float, float])

The translate(Tuple[float, float, float]) function adjusts the position of the geometry by a specified input.

Parameters

  • offset: A list or tuple representing the x, y, z offset values.

Return Value

This function returns None but modifies the geometry itself.

Example Usage

Geometry.translate([1.0, 0.0, 0.0])

Scale

Scale(self, scalar: float)

The Scale(self, scalar: float) function scales the size of the geometry by a specified factor.

Parameters

  • scalar: A float representing the scale factor.

Return Value

This function returns The geometry itself but modifies the size.

Example Usage

mesh.Scale(1.5)

Simulation Module

The Simulation Module handles various simulation operations.

Run Simulation Function: run_simulation()

The run_simulation() function starts the simulation process.

Parameters

  • duration: An integer representing the duration of the simulation in seconds.

Return Value

This function returns the results of the simulation.

Example Usage

results = run_simulation(10)
